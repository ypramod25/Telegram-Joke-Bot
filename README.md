## Setup Instruction 
1. **Clone the Repository**: Start by cloning the repository to your local machine using:
   ```bash
   git clone <https://github.com/ypramod25/Telegram-Joke-Bot>
   cd TelegramBot
   ```
2. **Install Dependencies**: Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```
3. **Create a .env File**: Create a `.env` file in the root directory and add your Telegram Bot Token:
   ```bash
   BOT_TOKEN=your_bot_token_here
   ```
4. **Run the Bot**: Start the bot using:
   ```bash
   node index.js
   ```
